## Tech Stack

**Client:** React, Redux, context API, CSS

**Deployment:** firebase

## Installation

Install yarn into your system

```bash
  npm install --global yarn
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/ratnakar5938/Whatsapp-clone.git
```

Go to the project directory

```bash
  cd Whatsapp-clone
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
